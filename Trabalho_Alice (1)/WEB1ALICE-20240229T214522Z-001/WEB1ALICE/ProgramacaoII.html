Programação II

A disciplina de Programação II, administrada pelo professor Bruno Vicente, atual coordenador do curso, vai ser abordada com a apresentação da Linguagem C. A seguir, vamos conhecer um pouco sobre essa linguagem, tão utilizada pelos programadores.


  • Linguagem C:
    •
    É uma linguagem de alto nível genérica, na qual, pode ser usada em diversos tipos de projeto, como a criação de aplicativos, sistemas operacionais, drivers, entre outros. Trata-se de uma linguagem estruturada que se tornou muito popular nos anos 80, tanto que é difícil encontrar arquiteturas para as quais não existam compiladores para a C, o que garante o seu elevado nível de portabilidade.

  • Principais características:
    •
    1. Portabilidade;
    2. Geração de código eficiente;
    3. Simplicidade;
    4. Confiabilidade;
    5. Facilidade de uso;
    6. Regularidade.
