Programação Web I

Resumo da disciplina:

Em resumo a disciplina de Programação Web I, aborda os conceitos básicos necessários para
desenvolver e projetar páginas web estáticas e interativas. Tem como objetivo introduzir os alunos
aos fundamentos do desenvolvimento web, focando principalmente em três tecnologias essenciais:
HTML (HyperText Markup Language), CSS (Cascading Style Sheets) e JavaScript

Conteúdos abordados até o momento:

  • Introdução ao HTML como a linguagem padrão para criação de páginas web.
  • Estudo dos elementos básicos do HTML, como tags, atributos, estrutura de página, links, imagens, listas e tabelas
  • Criação de paginas simples apenas com html
